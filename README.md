# AtlasStay

AtlasStay is a luxury shortlet and real estate marketplace focused on African cities.

The platform allows:
- Hosts to list luxury apartments
- Guests to book shortlet stays
- Diaspora investors to discover property opportunities
- Travelers to request concierge services like airport pickup and private drivers

## Core Cities
- Lagos
- Abuja
- Port Harcourt
- Accra

## Core Features
- Apartment listings
- Instant booking
- Host dashboards
- Concierge services
- Property investment marketplace

## Technology Stack
Frontend: Next.js
Backend: Supabase
Database: PostgreSQL
Payments: Paystack & Flutterwave
Hosting: Vercel
